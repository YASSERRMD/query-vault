# Todo Directory Structure

This folder contains development task tracking for the Rust project.

## Subfolders

| Folder | Purpose |
|--------|---------|
| features/ | New feature requests and implementation plans |
| bugs/ | Bug reports and fixes to track |
| ideas/ | Brainstorming and exploratory concepts |
| refactoring/ | Code improvement and cleanup tasks |
| documentation/ | Docs to write or update |

## Usage

Create .md files in each folder to track tasks. Example naming:
- features/001-add-query-caching.md
- bugs/fix-memory-leak.md
